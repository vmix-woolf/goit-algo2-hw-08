# goit-algo2-hw-08

Домашнє завдання з курсу Algorithms Part 2.

Проєкт містить два окремі завдання, присвячені роботі з алгоритмами
оптимізації доступу до даних та керування частотою запитів.

## Структура проєкту

- `lru_cache/` — Завдання 1  
  Оптимізація підрахунку сум діапазонів у великому масиві
  з використанням LRU-кешу.  
  Детальний опис — у `lru_cache/README.md`.

- `sliding_window/` — Завдання 2  
  Реалізація rate limiter для чат-системи
  на основі алгоритму Sliding Window.  
  Детальний опис — у `sliding_window/README.md`.

## Запуск

Кожне завдання запускається окремо зі своєї папки:

```bash
python lru_cache/main.py
python sliding_window/main.py
